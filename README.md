## Personal Expense and Budget Tracker

A simple, beginner-friendly Python project that helps users record, view, delete, and analyze daily expenses.
The program uses a CSV file to store entries and includes for category-wise and monthly summaries.

This project demonstrates file handling, data structures, user input processing in Python.

- View Expenses :

- Delete Expense :
Remove an entry using its Index number

- Summary :
Shows:
- Total amount spent
- Spending categorized by type (Food, Travel, Misc, etc.)

- Avoiding File Related Errors :
The program automatically creates the CSV file if it's missing.
