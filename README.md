## Personal Expense and Budget Tracker

A simple, beginner-friendly 12th-grade Python project that helps users record, view, delete, and analyze daily expenses.
The program uses a CSV file to store entries and includes optional matplotlib graphs for category-wise and monthly summaries.

This project demonstrates file handling, data structures, user input processing, and basic data visualization in Python.

‚ú® Features
- Add Expense :
Store date, category, and amount into a CSV file (manager.csv).

- View Expenses :
Display all stored expenses with running total.

- Delete Expense :
Remove an entry using its Index number

- Summary :
Shows:
- Total amount spent
- Spending categorized by type (Food, Travel, Misc, etc.)

- Plots (Optional) :
If matplotlib is installed:
- Category-wise bar chart
- Monthly spending line chart

- Avoiding File Related Errors :
The program automatically creates the CSV file if it's missing.
